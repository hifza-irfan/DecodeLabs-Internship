# ✒️ Calligraphy by Hifza — Sukoon Collection

A portfolio website for a calligraphy artist, showcasing Arabic and Urdu calligraphy artwork, services offered, and a way for visitors to get in touch.

## 📖 Overview

Sukoon Collection is a single-page portfolio built around the theme "finding the calm in the chaos." It includes a hero section, a services overview, an image gallery of calligraphy pieces, an artist's note, and a contact form.

## ✨ Features

- Sticky Header with Logo and Navigation
- Smooth Scroll Anchor Links
- Services Grid with Icon Cards
- Responsive Gallery Grid (3 / 2 / 1 Columns)
- Hover Animations on Service and Art Cards
- Contact Form with Name, Email, and Message Fields
- Direct Email Fallback Link
- Footer with Social Icons and Quick Links
- Back-to-Top Button

## 🛠️ Tech Stack

- HTML5
- CSS3 (Flexbox, Grid, Media Queries)
- Font Awesome (Social Icons)
- Google Fonts (Fredoka One, Play)

## 🚀 Running the Project

1. Clone or download the project folder.
2. Make sure the `images/` folder sits alongside `index.html` with the logo and all artwork images.
3. Open `index.html` directly in a browser, or run it through VS Code's Live Server.

No build steps, no dependencies — it's plain HTML and CSS.

## 🎨 Design Notes

- Warm Earthy Palette (Browns, Cream, Terracotta)
- Card-Based Layout for Services and Gallery
- Grid for Layout, Flexbox for Components
- Mobile Breakpoints at 900px and 600px

## ⚠️ Known Limitations

- Contact Form Has No Backend
- Social Media Links Are Placeholders
