# ☕ Buttercup Cafe

A responsive landing page for a fictional cafe, built as Project 2 for DecodeLabs. The focus of this project was practicing mobile-first design — writing CSS for small screens first, then layering in complexity for tablets and desktops with media queries.

## 📖 Overview

Buttercup Cafe is a single-page site with a hero video, a menu grid, an about section, opening hours, a reservation form, a photo gallery, and contact details. The tagline says it best: "Where every day feels like Sunday morning."

## ✨ Features

* Sticky Navbar with JavaScript Hamburger Menu
* Autoplaying Hero Video
* Responsive Menu Grid (1 / 2 / 3 Columns)
* Fluid Typography with clamp()
* Hover Animations on Cards and Gallery
* Reservation Form (Front-End Only)
* Photo Gallery Section
* Mobile-First Responsive Layout

## 🛠️ Tech Stack

* HTML5
* CSS3 (Flexbox, Grid, Custom Properties)
* Vanilla JavaScript

## 🚀 Running the Project

1. Clone or download the project folder.
2. Make sure the `images/` folder sits alongside `index.html` with all the referenced photos and the cafe video.
3. Open `index.html` directly in a browser, or run it through VS Code's Live Server for the smoothest experience.

No build steps, no dependencies — it's plain HTML, CSS, and JS.

## 🎨 Design Notes

* Mobile-First CSS with min-width Breakpoints
* Warm Color Palette (Orange, Cream, Olive, Brown)
* 44-48px Minimum Tap Targets
* Custom Hamburger Menu (No Popover API)

## ⚠️ Known Limitations

* Reservation Form Has No Backend
* Map Section Uses a Static Image
