# 🎯 Mood & Focus Dashboard

A minimalist productivity dashboard built with Vanilla JavaScript using the IPO (Input → Process → Output) pattern.

## 📖 Overview

Mood & Focus Dashboard helps users set their daily mood and stay productive. Users can select different moods, change the theme between light and dark mode, and save their preferences using localStorage for a personalized experience.

## ✨ Features

* 8 Mood Options (Focus, Calm, Energetic, Creative, Grateful, Motivated, Peaceful, Playful)
* Dark/Light Theme Toggle
* Dynamic Quotes and Accent Colors
* Active Mood Highlighting
* LocalStorage Support
* Responsive Design for Mobile, Tablet and Desktop

## 🛠️ Technologies Used

* HTML5
* CSS3
* Vanilla JavaScript
* LocalStorage API

## 🚀 Installation

### Clone Repository

### Navigate to Project Folder

```bash

```

### Run the Project

Open `index.html` in your browser or use Live Server in VS Code.

## 🧠 IPO Pattern

* **Input:** User selects a mood or toggles the theme.
* **Process:** JavaScript updates the application state.
* **Output:** The interface updates with new colors, quotes, and visual feedback.

## 📱 Responsive Design

The dashboard is fully responsive and works smoothly across desktop, tablet, and mobile devices.

## 🔒 Security

* No use of innerHTML
* Uses textContent for safe content updates
* No external scripts
